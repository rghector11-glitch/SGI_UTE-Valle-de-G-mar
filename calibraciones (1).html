Criterio por instrumento

El umbral de cada instrumento define el límite de aceptación para ese sensor concreto. Se edita en la ficha del instrumento. Si está vacío, se aplica el umbral global.

¿Cómo se calcula la desviación?

Desviación absoluta = Lectura equipo − Valor patrón

Desviación relativa (%) = (Desviación absoluta / |Valor patrón|) × 100

El sistema aplica el criterio de aceptación sobre el valor absoluto de la desviación relativa.

Ejemplo: Sonda pH, patrón = 7.00, lectura = 7.12

Desviación = 7.12 − 7.00 = +0.12 pH

Desv. % = (0.12 / 7.00) × 100 = +1.71 % → ✔ Aceptable si umbral ≥ 2 %
